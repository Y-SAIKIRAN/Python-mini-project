# Number Guessing Game 🎯

A simple Python number guessing game where the computer randomly selects a number between 1 and 100, and the player tries to guess it.

## Features

- Random number generation between 1 and 100
- Input validation (accepts only numbers)
- Hints for higher or lower guesses
- Option to quit the game anytime
- Simple command-line interface

## How It Works

1. The computer generates a random number between 1 and 100.
2. The player enters a guess.
3. The program:
   - Congratulates the player if the guess is correct.
   - Gives a hint if the guess is too high or too low.
4. The game continues until the correct number is guessed or the player types `quit`.

## Requirements

- Python 3.x
